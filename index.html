Thanh Dinh Ta

About me

I am a postdoctoral researcher at Joseph Fourier University, working in PACSS group at Verimag laboratory. I currently participate in BinSec project to design and implement a guided dynamic symbolic execution engine, and for a formal approach to deobfuscate virtualization-obfuscated binaries.

Before joining PACSS, I have completed my PhD at Lorraine University, in CARTE group at Loria laboratory. My thesis is about a formal model for obfuscating transformations on behaviors of programs, and about a semantics-based method to synthesizing hidden automata of input message parsers of malwares.

My research focuses on automated methods for computer security, particularly for binary code analysis and malware detection. I am interested in formal methods on program analysis/verification, and their applications in extracting high-level similarities between obfuscated programs.

Publications [DBLP]

ISSTA 2016
Specification of Concretization and Symbolization Policies in Symbolic Execution (accepted) (pdf)

Robin David, Sébastien Bardin, Laurent Mounier, Marie-Laure Potet, Thanh Dinh Ta, Jean-Yves Marion and Josselin Feist

This is a joint work with BinSec's members. The paper introduces a meta-language implemented in BINSEC/SE framework to specify explicitly concretization/symbolization policies for concolic execution. We show that implicit policies used in other frameworks can be implemented using our language. We presents the effect of applying different policies in symbolic execution on several large testsuites.

SANER 2016
BINSEC/SE: A Dynamic Symbolic Execution Toolkit for Binary-level Analysis (accepted) (pdf)

Robin David, Sebastien Bardin, Mounier Laurent, Thanh Dinh Ta, Jean-Yves Marion, Josselin Feist and Marie-Laure Potet

This is a joint work with BinSec's members. The paper introduces a framework for concolic (i.e. dynamic symbolic) execution on binary codes with parametric concretization/symbolization policies and an extensible execution engine based on the framework. We presents experimentations of BinSec's platform on security challenges and malwares.

FPS 2014
Malware Message Classification by Dynamic Analysis (pdf)

Guillaume Bonfante, Jean-Yves Marion, and Thanh Dinh Ta

This is a joint work my PhD advisors. The paper presents a mathematical approach to synthesizing the automaton used in the input message parser of a program. We need very few hypotheses on the actual implementation of the parser by the program and the soundness of the approach is rigrously proved. The experimental implementation given in PathExplorer has used automatic dynamic binary instrumentation; yet it can be made better by concolic execution.

TAMC 2014
A Categorical Treatment of Malicious Behavioral Obfuscation (pdf)

Romain Péchoux and Thanh Dinh Ta

This is a joint work with my first PhD co-advisor. The paper presents at the first time a formal model for obfuscating transformations on behaviors of programs. We then proves several constructive obfuscating transformations which can be extracted from the model. The experimental implementation is given in SyscallTrace. A much better implementation is then realized by FEEBO in Technische Universität München.
